1. Understanding Split Screen#

Split Screen feature has been available on Android phones since Android 7.0 Nougat. With the release of Android 10, this feature has been highly improved, making it easier to use and have better user experience. To use this feature, one must first know how it works. To get started with the split screen feature on Android 10, users can follow these steps: First, swipe up from the bottom of the screen to access the recent apps list. Once the list of recent apps appears, select the two apps you would like to open. After that, drag and drop one app to the top of the screen and the other app to the bottom. This will activate split screen and both chosen apps will be open at the same time. Finally, users can adjust the size of the apps by dragging the divider in the middle of the screen. With this feature, users can maximize their phone’s multitasking capabilities.
